# ARMA3WARFARE_SERVER
Server scripts

## Project Structure

````
waspserver
	Base            - faction base management tools
	Commander       - base Commander related functions
	Economy         - money & suppluy utility functions
	GearTemplates   - a set of functions for gear templates management
	HeadlessClient  - a set of functions for headless client management
	Module          - extra modules to control weather conditions, vehicle features
	MySQL           - database related functions
	Player          - tools to control player entity in game
	Task            - in game task management
	Team            - a set of functions for operations on team object
	Unit            - unit related control functions
	Warfare         - game mode related functions

````
